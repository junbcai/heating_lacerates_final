# Thermal stress accelerates asexual development and alters symbiosis and host physiology in a model cnidarian-algal symbiosis

This repository contains the data, analysis scripts, figures, and supplementary materials associated with the manuscript:

**Cai, J. et al.**  
*Thermal stress accelerates asexual development and alters symbiosis and host physiology in a model cnidarian-algal symbiosis*

The study investigates how elevated temperature influences asexual development, symbiosis dynamics, and host physiology in the sea anemone *Exaiptasia diaphana* (Aiptasia), a model system for studying cnidarian–algal symbiosis.

---

## Project Overview

Climate-driven ocean warming destabilizes cnidarian–algal symbioses and contributes to bleaching across coral reef ecosystems. While thermal stress responses have been extensively studied in adult cnidarians, comparatively less is known about how elevated temperature affects early asexual developmental stages.

Using pedal lacerates of *Exaiptasia diaphana*, this study examines the effects of ambient (25°C) and heat stress (32°C) conditions on:

- Tentacle development
- Respiration
- Symbiont density
- Host survival
- Cellular physiology

The repository contains the full computational workflow used to process and analyze the experimental datasets associated with the manuscript.
